TalkDoc - AI Medical Assistant Chatbot

Overview

TalkDoc is an AI-powered web application that serves as a medical assistant chatbot. By utilizing speech recognition and natural language processing (NLP), the tool allows users to describe their symptoms via speech and receive potential over-the-counter medication recommendations and health advice. Developed using Streamlit and powered by Azure Cognitive Services and OpenAI GPT-3, TalkDoc offers a multilingual, intuitive, and efficient experience for healthcare self-assistance.

Key Features

  • Speech-to-Text: Converts user speech into text using Azure’s Speech SDK.
  • AI-Powered Medical Advice: Utilizes OpenAI GPT-3 via Azure to generate relevant and accurate health advice based on symptoms.
  • User-Friendly Interface: Built with Streamlit, offering a clean, responsive, and multilingual interface.
  • Real-Time Interaction: Fast processing and response for near-instant medical suggestions.
  • Multilingual Support: Supports English, Hindi, Spanish, French, Russian, and Japanese.

Technical Stack

  • Frontend: Streamlit
  • Backend: Azure OpenAI API, AssemblyAI
  • Programming Language: Python
  • APIs: Azure Cognitive Services Speech API, OpenAI GPT-3

How It Works

  1. The user clicks “Say Symptoms” to record their voice input.
  2. Speech is transcribed using Azure's Speech SDK.
  3. The transcribed text is processed and sent to OpenAI’s model via a prompt.
  4. OpenAI generates contextual medical advice based on the provided symptoms.
  5. The response is displayed in a user-friendly format.

System Design & Testing

The system was built with robust error handling, efficient API integration, and clear UI components. It underwent unit, integration, system, and user acceptance testing. Testing confirmed functionality, performance under load, and responsiveness to various language inputs.

Results & Discussion

  • Speech Recognition: High accuracy, with some variation across languages. Noise interference remains a challenge.
  • Medical Advice: Relevance and coherence were consistently rated high by testers. Suggestions were contextually appropriate and easy to understand.
  • Interface Usability: Users found the layout intuitive and responsive, with special praise for the multilingual functionality.

Challenges & Improvements

  • Enhancing recognition accuracy in noisy environments.
  • Expanding the medical advice database for deeper context.
  • Adding user profile support for personalized health tracking and history-based advice.

Future Directions

Plans for TalkDoc include integrating a comprehensive medical database, improving background noise handling, and introducing personalized user experiences. Additionally, the team aims to refine its prompt engineering for more detailed and specific advice across medical conditions.
